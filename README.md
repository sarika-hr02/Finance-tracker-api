# Finance Tracker API (Flask)

## Description
This is a simple Flask-based backend project to manage income and expenses.

## Features
- Add (POST /add)
- Get (GET /get)
- Update (PUT /update/<index>)
- Delete (DELETE /delete/<index>)
- Summary (GET /summary)
- Filter (GET /filter/income, /filter/expense)

## Run
python app.py
